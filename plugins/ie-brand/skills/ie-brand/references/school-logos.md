# School Logo URLs

Each school has a logo family with the following variants:
- **Default** — standard on-brand version (for light backgrounds)
- **Accent negative** — school accent on dark/Ocean Blue backgrounds
- **Full color** — full-colour version
- **Full color negative** — full-colour inverted for dark backgrounds

Each variant comes in two orientations:
- **Hztl** (horizontal) — for headers, navbars, wide containers
- **Vertical / stacked** — for compact spaces, mobile, and square contexts

Always prefer **SVG**; use PNG as fallback.
Apply the same contrast rules as the corporate logo: use negative/inverted variants whenever the background is dark or Ocean Blue.

## Variant Selection Guide

| Background | Recommended variant |
|---|---|
| White / Light grey / IE White | Default |
| Ocean Blue / Dark section | Accent negative or Full color negative |
| School accent color | Full color negative or Full White |
| Printed / full-color document | Full color |
